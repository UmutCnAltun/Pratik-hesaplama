# 🧮 Pratik Hesaplama

Pratik Hesaplama, günlük hayatta ihtiyaç duyulan temel matematiksel ve finansal hesaplamaları tek bir çatı altında toplayan, kullanıcı dostu ve hızlı bir web aracıdır. Karmaşık arayüzlerden uzak, tamamen işlevselliğe odaklanan bir yapıya sahiptir.

## 🚀 Öne Çıkan Özellikler

* Çeşitli Hesaplama Modülleri: KDV Hesaplama, Vize-Final Notu, İndirim Hesaplama vb.

* Hızlı Sonuç: Sayfa yenilemeye gerek kalmadan, anlık hesaplama altyapısı.

* Responsive Tasarım: Hem masaüstü hem de mobil cihazlarla tam uyumlu arayüz.

## 🛠️ Kullanılan Teknolojiler

Bu proje, herhangi bir kütüphane veya framework bağımlılığı olmadan Vanilla Stack ile geliştirilmiştir:

* HTML5: Sayfa yapısı ve semantik içerik.

* CSS3: Modern ve temiz arayüz tasarımı (Flexbox/Grid).

* JavaScript: Hesaplama mantığı ve dinamik DOM yönetimi.

## 📥 Kurulum ve Çalıştırma

Projeyi klonlayın:

```
git clone https://github.com/UmutCnAltun/Pratik-hesaplama.git
```

Dizine gidin:

```
cd Pratik-hesaplama
```

Çalıştırın:

**index.html dosyasını herhangi bir tarayıcıda açmanız yeterlidir.**

## 📖 Neden Pratik Hesaplama?
İnternet üzerindeki birçok hesaplama sitesi reklamlar ve karmaşık menülerle doludur. Bu projenin amacı, kullanıcının aradığı hesaplama aracına saniyeler içinde ulaşmasını ve sonucu en temiz şekilde almasını sağlamaktır.
